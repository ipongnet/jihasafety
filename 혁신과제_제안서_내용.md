## ❶ 과제 개요

### ○ 추진 배경

- 도로굴착 공사 시 지하매설물(가스, 상·하수도, 통신, 전력 등) 손상 방지를 위해 유관기관 사전 협의가 법적으로 의무화되어 있으나, 현행 협의 절차가 수기·대면 기반으로 운영되어 비효율이 발생
  * 시공업체는 해당 지역 담당 부서를 직접 파악하여 전화·팩스·이메일로 개별 연락해야 하며, 담당자 부재 시 협의 지연 빈발
  * 연간 약 OO건의 협의 요청 중 담당자 연결 실패·누락으로 인한 지연이 평균 O일 소요
- 정부 공공AX(AI 전환) 정책 기조에 따라, 반복적 행정 절차를 디지털 자동화하여 업무 효율성과 대국민 서비스 품질을 동시에 향상시킬 필요

### ○ 현황 및 문제점

- (비효율적 수기 프로세스) 시공업체가 굴착 위치의 관할 지자체·유관기관 담당자를 수작업으로 파악해야 하며, 부서 변경·담당자 이동 시 최신 정보 확보가 어려움
  * 담당자 1인당 일 평균 O건의 전화·팩스 문의 응대에 업무시간 소요
- (저촉여부 회신 지연) 지하시설물 저촉여부 판정 결과를 수기로 작성·발송하여 회신까지 평균 O일 소요
  * 회신 누락·지연으로 공사 착공이 지체되는 사례 발생
- (이력 관리 부재) 협의 요청·회신 이력이 체계적으로 관리되지 않아, 사후 추적·통계 분석이 불가능
  * 동일 구간 중복 협의, 미회신 건 추적 등 관리 사각지대 존재
- (정보 접근성 부족) 시공업체가 해당 지역 지하매설물 담당자 정보에 접근하기 어려워 초기 연락 단계에서 상당한 시간 소모

### ○ 개선방향

- (원스톱 자동 협의 체계) 시공업체가 웹 플랫폼에서 공사 정보를 입력하면 주소 기반으로 담당자를 자동 매칭하여 협의 요청 이메일을 즉시 발송하는 자동화 시스템 구축
  * 주소 입력 → 지오코딩(좌표 변환) → 관할 담당자 자동 매칭 → 이메일 자동 발송의 전 과정을 웹 기반으로 일원화
- (저촉여부 자동 회신) 내부 시스템에서 저촉여부 판정 완료 시 API 연동을 통해 접수자에게 결과(PDF 포함)를 자동 이메일 발송
  * 수기 회신 절차 제거, 판정 완료 즉시 실시간 회신
- (통합 이력 관리) 접수번호 체계 도입 및 전 과정 이력 DB화로 현황 파악·통계 분석 기반 마련
  * 접수, 발송, 회신, 저촉여부까지 전 생애주기 추적 가능
- (담당자 정보 공개) 지역별 지하매설물 담당자 현황을 공개 페이지로 제공하여 시공업체의 정보 접근성 향상

### ○ 세부 추진계획

- (1단계: 시범 구축 — 2026년) 웹 기반 협의 요청 자동화 플랫폼 개발 및 1개 지역본부 시범 운영
  * 굴착 요청 접수 폼, 주소 기반 담당자 자동 매칭, 이메일 자동 발송 기능 구현
  * Daum Postcode API(주소 검색) + V-World API(지오코딩) + Leaflet(지도 시각화) 연동
  * 접수번호 자동 생성 체계 구축 (예: 경기분당-2026-0001)
  * 관리자 대시보드: 접수 이력 조회, 상태 관리, 담당자 CRUD
- (2단계: 자동 회신 체계 — 2026~2027년) 내부 저촉여부 판정 시스템과 API 연동하여 자동 회신 기능 구축
  * REST API 기반 연동: 판정 결과(저촉/비저촉, 멘트, PDF) 자동 수신 → 접수자 이메일 자동 발송
  * 관리자 페이지에서 회신 현황·저촉여부 실시간 모니터링
- (3단계: 전사 확대 — 2027년) 전 지역본부·지사로 확대 적용 및 고도화
  * 전국 지자체 담당자 DB 확충 (현재 OO개 → 전체 시·군·구 커버)
  * 대국민 소통 강화: 협의 진행 상황 실시간 조회 기능 제공
  * 정책소통창구 운영: 시공업체 피드백 수집 → 시스템 개선에 반영
  * 통계 분석 대시보드 고도화 (지역별·기간별·저촉률 등)

| 시행 전 | 시행 완료 후 |
|---------|------------|
| 시공업체가 관할 담당자를 직접 파악·연락해야 하며, 전화·팩스 기반 수기 협의로 평균 O일 소요. 저촉여부 회신 누락·지연 빈발. 이력 관리 불가. | 웹 플랫폼에서 원클릭 협의 요청, 담당자 자동 매칭·이메일 즉시 발송. 저촉여부 판정 즉시 자동 회신. 접수번호 기반 전 과정 이력 추적·통계 분석 가능. |

### ○ 기대효과

- (업무 효율 향상) 담당자 수기 응대 업무 80% 이상 절감, 협의 요청~회신 완료까지 소요 기간 대폭 단축
  * 기존 평균 O일 → 자동 회신 도입 후 당일 처리 가능
- (대국민 서비스 개선) 시공업체의 협의 절차 진입 장벽 해소, 24시간 웹 접수 가능으로 시간·장소 제약 제거
  * 담당자 현황 공개로 정보 비대칭 해소
- (안전사고 예방) 협의 누락·지연에 따른 지하매설물 손상 사고 위험 감소
  * 자동 매칭·자동 발송으로 협의 누락 원천 방지
- (데이터 기반 정책 수립) 지역별·시기별 협의 현황, 저촉률 등 통계 데이터 축적으로 시설물 관리 정책 수립 기초 자료 확보
- (디지털 전환 선도) 공공AX 정책 기조에 부합하는 대표적 업무 자동화 사례 창출

## ❸ 관련 법령 및 지침 개정계획

- (도로법 제52조) 도로굴착 시 지하매설물 관리기관과의 사전 협의 의무 — 현행 유지, 협의 방식의 디지털 전환 추진
  * 전자적 협의 요청·회신을 유효한 협의 절차로 인정하는 내부 지침 마련 필요
- (지하안전관리에 관한 특별법) 지하시설물 정보 관리 및 안전관리 — 자동화 시스템의 법적 근거 검토
  * 시스템을 통한 협의 이력을 공식 기록으로 인정하기 위한 지침 개정 검토
- (개인정보보호법) 담당자 정보(이름, 이메일, 전화번호) 수집·이용 동의 절차 및 보호 조치
  * 담당자 정보 등록 시 동의서 양식 마련, 암호화 저장·접근 통제 적용
